# Portfólio Pessoal — João Pedro


> Portfólio pessoal criado para apresentar projetos, habilidades e contatos profissionais de João Pedro.

## ✨ Sobre o Projeto

Este projeto é um site de portfólio moderno e responsivo, desenvolvido com foco em apresentação visual, usabilidade e design limpo.

## 🚀 Funcionalidades

- Layout responsivo para desktop e mobile
- Navegação fixa com menu hambúrguer em telas pequenas
- Seção "Sobre" com histórico e objetivo profissional
- Seção "Skills" com tecnologias estudadas e domínio
- Seção "Projetos" destacando trabalhos já concluídos
- Seção "Contato" com formulário de mensagem
- Animações suaves de fade-in e scroll reveal

- Áreas de conteúdo bem separadas por divisores
- Botões e cards com hover suave
